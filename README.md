# Telegram Bot

Многофункциональный Telegram бот на aiogram 3.x с базой данных, админкой и статистикой.

## ⚠️ Требования

- **Python 3.11 или 3.12** (НЕ 3.14!)
- Windows 10/11
- Токен бота от @BotFather

## 🚀 Быстрая установка

### Шаг 1: Установите Python 3.12

**Вариант A: Автоматическая установка**
```batch
install_python312.bat
```

**Вариант B: Ручная установка**
1. Скачайте: https://www.python.org/downloads/release/python-3124/
2. Запустите установщик
3. ✅ Отметьте "Add Python 3.12 to PATH"

### Шаг 2: Настройте бота

```batch
setup.bat
```

Отредактируйте `.env` файл:
```env
BOT_TOKEN=ваш_токен_от_BotFather
ADMIN_IDS=ваш_id_от_userinfobot
```

### Шаг 3: Запустите бота

```batch
run.bat
```

## 📋 Команды бота

### Пользовательские
- `/start` - Запуск бота

- `/stats` - Личная статистика
- `/settings` - Настройки
- `/about` - О боте
- `/support` - Поддержка
- `/faq` - Часто задаваемые вопросы

### Административные
- `/admin` - Админ-панель
- `/users` - Список пользователей
- `/stats_all` - Общая статистика
- `/broadcast` - Рассылка
- `/block` - Блокировка
- `/unblock` - Разблокировка

## ❗ Решение проблем

### Python 3.14+ not supported
**Проблема:** У вас Python 3.14, который не поддерживается библиотеками.

**Решение:**
```batch
install_python312.bat
```

### Failed building wheel for pydantic-core
**Проблема:** Попытка компиляции Rust пакетов на неподдерживаемой версии Python.

**Решение:** Установите Python 3.12 (см. выше)

### BOT_TOKEN is not set
**Проблема:** Не настроен токен бота.

**Решение:**
1. Получите токен от @BotFather
2. Откройте `.env` файл
3. Установите `BOT_TOKEN=ваш_токен`

### Бот не отвечает
**Проверьте:**
1. Консоль открыта и нет ошибок
2. Токен правильный в `.env`
3. Пишете боту в ЛИЧКУ, а не в группу
4. Логи в папке `logs/`
