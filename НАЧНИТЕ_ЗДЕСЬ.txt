# ⚠️ ВАЖНАЯ ИНСТРУКЦИЯ ПО ЗАПУСКУ БОТА

## Проблема
У вас установлен **Python 3.14**, который НЕ поддерживается библиотеками aiogram и pydantic.

## ✅ Решение

### Вариант 1: Автоматическая установка Python 3.12 (Рекомендуется)

1. Запустите в папке TelegramBot:
   ```
   install_python312.bat
   ```

2. После установки Python 3.12, запустите:
   ```
   setup.bat
   ```

3. Если setup.bat пройдёт успешно, запускайте бота:
   ```
   run.bat
   ```

### Вариант 2: Ручная установка Python 3.12

1. Скачайте Python 3.12.4:
   https://www.python.org/ftp/python/3.12.4/python-3.12.4-amd64.exe

2. Запустите установщик
   ✅ Обязательно отметьте "Add Python 3.12 to PATH"

3. После установки откройте НОВОЕ окно командной строки

4. Проверьте версию:
   ```
   python --version
   ```
   Должно быть: Python 3.12.x

5. Перейдите в папку TelegramBot и запустите:
   ```
   setup.bat
   ```

## 📋 Что делать после установки

1. Откройте `.env` файл (создастся после setup.bat)

2. Получите токен бота:
   - Telegram → @BotFather
   - Команда: /newbot
   - Следуйте инструкциям
   - Скопируйте токен

3. Получите свой ID:
   - Telegram → @userinfobot
   - Отправьте любое сообщение
   - Скопируйте ID

4. Вставьте в `.env`:
   ```
   BOT_TOKEN=ваш_реальный_токен_без_кавычек
   ADMIN_IDS=ваш_id_числом
   ```

5. Запустите бота:
   ```
   run.bat
   ```

## ❌ Что НЕ делать

- НЕ пытайтесь запустить с Python 3.14 - не будет работать
- НЕ добавляйте кавычки в .env файл
- НЕ закрывайте консоль пока работает бот

## 📞 Если что-то не работает

1. Проверьте версию Python: `python --version`
2. Должно быть 3.12.x (НЕ 3.14!)
3. Проверьте логи в папке `logs/`
4. Запустите `test.bat` для диагностики
